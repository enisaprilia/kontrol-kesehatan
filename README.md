# kontrol-kesehatan
cek kesehatan di rumah sakit
sebelum pemeriksaan:
buat janji temu 
siapkan dokumen
catat pertanyaan
saat pemeriksaan:
registrasi 
pengukuran tanda vital 
konsultasi dengan dokter
pemeriksaan fisik
pemeriksaan penunjang (jika diperlukan)
pembayaran
setelah pemeriksaan:
ambil hasil pemeriksaan
konsultasi lanjutan 
catat hasil pemeriksaan
